# 何哥的网站导航系统

一个基于 **WebStackPage** 二次开发的个人网址导航站，  
用于整理和分享常用的优质网站、工具与资源。

## ✨ 项目简介

本项目基于开源项目 **WebStackPage** 搭建，  
在原有基础上对 **Logo / 页面结构 / 交互 / 功能** 进行了二次开发与个性化定制。

主要用于个人学习、工作效率提升与资源沉淀。

- 支持 PC / 移动端自适应
- 页面简洁、分类清晰
- 持续维护与更新

## 🔧 二次开发内容

- 🎨 自定义 Logo 与站点名称
- 🧭 页面结构调整（About 页面重构）
- 🪟 站长信息卡片弹窗（动画 + 移动端适配）
- 📋 一键复制联系方式
- 🔼 返回顶部按钮优化
- 🔍 站内搜索功能（PC / 移动端通用）
- 🧩 细节动效与整体视觉统一

## 🧱 技术栈

- HTML5
- CSS3
- JavaScript / jQuery
- Bootstrap
- Font Awesome
- Xenon UI（WebStack 原模板）

## 📁 项目结构说明

.
├── index.html # 首页（导航主页面）
├── about.html # 关于页面（已二开）
├── assets/
│ ├── css/ # 样式文件
│ ├── js/ # JS 脚本
│ ├── images/ # 图片资源（Logo / 头像等）
│ └── fonts/ # 字体与图标
└── README.md

## 🚀 部署方式

该项目为纯静态页面，可部署在任意静态托管平台：

- GitHub Pages
- Vercel
- Netlify
- 服务器 + Nginx
